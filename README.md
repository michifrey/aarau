# Aarau – Info-Seite

Eine statische Informationsseite über die Stadt Aarau mit Fokus auf **Geschichte** und den traditionsreichen **Maienzug**.

Design in den **Stadtfarben Rot / Schwarz / Weiss**, angelehnt an den Auftritt von maienzug.info.

## Inhalt

- `index.html` – die Seite mit allen Abschnitten
- `style.css` – das Styling (modernes, responsives Layout in Rot/Schwarz/Weiss)
- `map.js` – die interaktive Karte (Leaflet)
- `images/` – Logo, Favicon und Platz für eigene Fotos (siehe `images/README.md`)

## Ansehen

Einfach `index.html` im Browser öffnen – keine Installation oder Build nötig.
(Karte und Schriften laden externe Inhalte und brauchen eine Internetverbindung.)

## Abschnitte

- **Überblick** – Aarau auf einen Blick (Lage, Altstadt, Identität)
- **Geschichte** – Zeitstrahl von der Gründung um 1240 bis zur Kantonshauptstadt
- **Karte** – interaktive Karte (Leaflet/OpenStreetMap) mit Markern und Fotos.
  Umschaltbar auf **historische Karten** (Dufour-/Siegfriedkarte ≈1850) von swisstopo.
- **Maienzug** – das wichtigste Fest der Stadt: Ablauf, Fakten und Tradition
- **Programm** – der Maienzug-Tagesablauf 2026 (Vorabend + Maienzug) für die Kids, inkl. Checkliste
- **Besuchen** – Tipps für einen Besuch

## Anpassen

- **Fotos** für die Karten-Marker: siehe `images/README.md`
- **Logo**: `images/logo.svg` durch das Original von maienzug.info ersetzen
- **Programm-Zeiten**: jährlich im Abschnitt `#programm` in `index.html` aktualisieren

> Hinweis: Historische Angaben und Programmzeiten dienen der Übersicht – bitte vor dem Fest
> mit den offiziellen Quellen der Stadt Aarau abgleichen.
