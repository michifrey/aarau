# Bilder

Hier kommen die Fotos für die Karten-Marker hin. Lege die Dateien mit genau diesen Namen ab,
dann erscheinen sie automatisch in den Popups (sonst wird ein Platzhalter angezeigt):

- `altstadt.jpg` – Altstadt / schöne Giebel
- `schloessli.jpg` – Schlössli / Stadtmuseum
- `stadtkirche.jpg` – Stadtkirche
- `kunsthaus.jpg` – Aargauer Kunsthaus
- `kantonsschule.jpg` – Alte Kantonsschule
- `bahnhof.jpg` – Bahnhof Aarau
- `schachen.jpg` – Schachen / Maienzug-Festplatz

Empfohlene Grösse: ca. 480×280 px (Querformat).

## Logo

- `logo.svg` – Maienzug-Aarau-Wortmarke (Platzhalter im Aarauer Stil, Rot/Schwarz/Weiss).
  Ersetze diese Datei durch das offizielle Logo von maienzug.info, sobald du es hast.
- `favicon.svg` – Browser-Icon (Blumen-Motiv).
